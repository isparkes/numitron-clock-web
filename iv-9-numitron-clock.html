Description

This is a really tiny desktop clock which will add vintage look to your desktop setup. Perforated wooden enclosure milled out of ash tree looks especially neat with sanded stainless cover.

The clock is built with 6 numeric Numitron tubes and 2 tiny filament colot tubes.

Built-in RGB led back-light (with 50 levels for each channel) allows you to set your favorite color.

Simple user-friendly menu (navigated with 3 buttons - menu and +/-) allows you to manage all the settings.

Time accuracy is provided with built-in RTC (Real Time Clock DS3231) module and backuped with CR2032 battery while the clock is powered off.

Current date is shown each 1-5-th minute in 3 various formats: DD:MM:YY, MM:DD:YY or YY:MM:DD. The current time could be configured to 12 or 24 hour format.

There is also three modes for colon tubes: 1) Blinks once a second (is set as a default option); 2) permanently OFF; 3) permanently ON.

Clock could be set to beep once an hour (when hour starts).

All setting are stored in non-volatile memory (settings are restored after power-offs).

Buying our product you get completely assembled, ready for use and well tested clock. DC power adapter and user manual are provided.

The enclosure has a following dimensions: height - 24 mm; width - 100 mm: lenght - 65 mm. Tube height is: 45 mm. Digits size depends on model of the tubes.
